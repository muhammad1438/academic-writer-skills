# Academic Language Toolkit

Academic discourse is characterised by a specialised lexicon, objective tonal registers, and syntactic structures that prioritise information density and conceptual precision over conversational fluidity.

---

## 1. Hedging Language (Epistemic Modality)

Hedging is **not weakness** — it is a marker of academic credibility. It signals that the writer is aware of the limits of the evidence and the boundaries of their claims. Overconfident assertions without hedging are a red flag in academic review.

### Modal Verbs (Degrees of Certainty)
| Verb | Certainty Level | Example |
|---|---|---|
| will | Very high | "This will result in..." |
| would | High / conditional | "This would suggest..." |
| should | Moderate / expected | "This should be interpreted..." |
| may | Moderate | "This may indicate..." |
| might | Lower | "This might be attributed to..." |
| could | Low / speculative | "This could be explained by..." |

### Hedging Adverbs
- **High certainty:** generally, typically, usually, commonly, predominantly, consistently
- **Moderate:** often, frequently, largely, broadly
- **Tentative:** apparently, seemingly, arguably, possibly, presumably, conceivably, perhaps

### Hedging Adjectives
- possible, probable, likely, unlikely, potential, apparent, plausible, uncertain

### Hedging Verbs
- suggest, indicate, appear, seem, tend, imply, propose, hypothesise, point to

### Hedging Phrases (Ready to Use)
- "Evidence suggests that..."
- "It is possible that..."
- "It could be argued that..."
- "There is reason to believe that..."
- "The data appear to indicate..."
- "It seems likely that..."
- "Under certain conditions, this may..."
- "These findings tentatively suggest..."
- "Preliminary evidence points to..."

### Discipline Differences
- **STEM:** Hedges primarily through statistical language ("p < 0.05," "within a 95% confidence interval") and passive constructions
- **Social Sciences:** Heavy use of modal verbs and tentative reporting verbs
- **Humanities:** Hedging through phrases like "arguably," "one might contend," "it is tempting to read this as"

## 2. Academic Verbs by Function

### Reporting / Introducing Others' Work (Neutral)
states, notes, writes, observes, reports, describes, presents, records, lists, mentions

### Positive / Supportive Reporting
demonstrates, confirms, proves, establishes, validates, supports, corroborates, verifies, shows, reveals, identifies, illustrates

### Tentative Reporting
suggests, implies, indicates, proposes, hypothesises, theorises, speculates, posits, infers, intimates

### Critical / Negative Reporting
challenges, disputes, contradicts, refutes, questions, criticises, contests, undermines, problematises, overlooks, neglects, fails to account for, underestimates, oversimplifies

### Conceding / Acknowledging
acknowledges, admits, concedes, recognises, accepts, grants, notes (in a limiting context)

### Emphasising
highlights, emphasises, stresses, underscores, foregrounds, draws attention to, points out

### Describing Research Process
analysed, examined, investigated, explored, studied, assessed, evaluated, measured, tested, surveyed, interviewed, reviewed, compared, contrasted, computed

### Presenting Your Own Argument
argue, contend, maintain, assert, claim, propose, demonstrate, show, illustrate, submit (formal), advance (a position)

### Structuring the Paper
describes, outlines, begins by, proceeds to, turns to, focuses on, concludes by, is structured as follows

## 3. Transition and Cohesion Devices

### Adding Information
furthermore, moreover, in addition, additionally, also, besides, equally, similarly, likewise, in the same vein, what is more

### Contrasting / Opposing
however, nevertheless, nonetheless, on the other hand, in contrast, conversely, yet, although, whereas, despite this, in spite of, while, by contrast, that said

### Conceding a Point
admittedly, granted, it is true that, although it may be argued that, while X may be the case, to be sure

### Cause and Effect
therefore, consequently, as a result, hence, thus, accordingly, for this reason, this means that, this leads to, it follows that

### Exemplification
for example, for instance, such as, including, specifically, in particular, namely, to illustrate, as an illustration, this is evident in

### Sequencing
first, second, third, subsequently, then, following this, next, finally, lastly, previously, prior to this, in the first instance

### Summarising / Concluding
in summary, in conclusion, to summarise, overall, on balance, in short, ultimately, taken together, in light of the above, these findings suggest

### Comparing
similarly, in the same way, likewise, by analogy, correspondingly, in parallel

### Emphasising
notably, significantly, crucially, importantly, above all, of particular relevance

## 4. Nominalization — Converting to Academic Noun Forms

Nominalization shifts dynamic processes into static conceptual nouns, enabling:
- Foregrounding the phenomenon rather than the actor
- Increased information density
- Greater objectivity and scholarly distance

**How it works:** "We investigated the effects" → "An investigation of the effects was conducted"

| Verb/Adjective | Nominalized Form |
|---|---|
| investigate | investigation |
| evaluate | evaluation |
| analyse | analysis |
| develop | development |
| implement | implementation |
| contribute | contribution |
| determine | determination |
| apply | application |
| assume | assumption |
| interfere | interference |
| demonstrate | demonstration |
| establish | establishment |
| identify | identification |
| interpret | interpretation |
| represent | representation |
| respond | response |
| significant | significance |
| intense | intensity |
| difficult | difficulty |
| effective | effectiveness / efficacy |
| similar | similarity |
| complex | complexity |
| diverse | diversity |
| accurate | accuracy |
| stable | stability |

**Caution:** Excessive nominalization buries agency, creates dense impenetrable prose, and is increasingly discouraged. Use it purposefully — not reflexively.

## 5. Academic Word List (AWL) — Sublist 1 (Most Frequent)

The AWL (Coxhead, Victoria University of Wellington) contains 570 word families covering ~10% of all words in academic texts. Combined with the 2,000-word General Service List, it provides ~90% lexical coverage of standard academic prose.

**Sublist 1 — Core 60 Word Families (learn these first):**
analyse, approach, area, assess, assume, authority, available, benefit, concept, consistent, constitutional, context, contract, create, data, definition, derived, distribution, economic, environment, established, estimate, evident, export, factors, financial, formula, function, identified, income, indicate, individual, interpretation, involved, issues, labour, legal, legislation, major, method, occur, percent, period, policy, principle, procedure, process, required, research, response, role, section, significant, similar, source, specific, structure, theory, variables

**Key rule:** Learn the full morphological family, not just the headword. Example:
- *analyse → analysed, analyser, analyses, analyst, analytic, analytical, analytically*

## 7. Precision and Quantifying Language

### Quantifiers
- Large amounts: a majority of, a substantial proportion, a considerable number, the bulk of
- Small amounts: a minority of, a small proportion, a negligible number, few
- Approximate: approximately, roughly, around, in the region of
- Exact: precisely, exactly, specifically

### Comparative Language
higher/lower than, greater/lesser than, more/fewer than, twice as, three times as, significantly higher, marginally lower, considerably more, substantially greater, markedly different, negligibly different, broadly comparable

### Degree/Intensity Modifiers (Academic-appropriate)
considerable, substantial, significant, notable, marked, pronounced, moderate, marginal, negligible, slight, minimal, extensive, comprehensive, limited

**Avoid:** very, really, a lot, tons, hugely, incredibly — replace with: considerably, substantially, significantly, notably, markedly

## 8. Replacing Informal Words

| Informal | Academic Alternative |
|---|---|
| a lot of | a considerable number of, a substantial proportion of |
| but | however, nevertheless, yet |
| also | furthermore, in addition, moreover |
| show | demonstrate, indicate, reveal, illustrate |
| use | utilise, employ, apply |
| help | facilitate, assist, support |
| change | alter, modify, transform |
| look at | examine, investigate, analyse |
| find out | determine, ascertain, establish |
| think | contend, propose, argue, maintain |
| get | obtain, acquire, yield |
| big/large | substantial, considerable, extensive |
| small | minimal, limited, marginal |
| etc. | and so forth / and similar phenomena (or list completely) |
| thing | phenomenon, factor, element, aspect, dimension |
| stuff | material, content, data, evidence |

## 9. Manchester Academic Phrasebank — Quick Reference Sections

The University of Manchester Phrasebank is the gold standard for content-neutral academic phrases. Key functional areas:

- **Introducing work:** "This paper examines...", "The aim of this study is to..."
- **Referring to literature:** "A number of scholars have argued...", "There is a growing body of literature that..."
- **Describing methods:** "Data were collected using...", "A qualitative approach was adopted..."
- **Presenting results:** "The results indicate that...", "Table 1 shows..."
- **Discussing findings:** "These findings suggest...", "This is consistent with the work of..."
- **Writing conclusions:** "This study has shown...", "Future research should..."

Full resource: https://www.phrasebank.manchester.ac.uk/
